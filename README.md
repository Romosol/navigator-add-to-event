# Navigator - add to event/Навигатор - создать заявку на новое мероприятие
**Скрипт, который добавляет детей на мероприятие в Навигаторе дополнительного образования**

## Установка

Для работы понадобится Python последней версии, библиотека Pandas для работы с таблицами .xclx и Selenium для управления браузером
Не забудьте выбрать при установке Pythin - Add python.exe to PATH

В командной строке введите:


`pip install selenium pandas openpyxl`


Для запуска введите:


`python NavAddToEvent.py`


При первом запуске вас попросят ввести логин, пароль, название мероприятия, дату и время
Введённые данные будут храниться в config.json

Программа автоматически открывает окно браузера, авторизируется на сайте и ждёт подтверждения для начала работы.

Шаги работы программы:
1. Открывает окно браузера
2. Авторизация
3. Нажимает кнопку создать
4. Вводит название мероприятия, дату, время
5. Вводит ФИО ребёнка, выбирает из выпадающего списка ребёнка с подходящими ФИО и датой рождения
6. Нажимает кнопку сохранить
7. Начинает заного с 3 шага

В файл list.xclx введите список детей и даты рождения, начиная со второй строки
Дата в формате дд.мм.гггг

В результате работы программа создаёт 2 лог файла

В конце работы программа закрывается
